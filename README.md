# 🌐 Cisco Packet Tracer — Networking Projects

A collection of hands-on networking labs built with Cisco Packet Tracer,
covering core CCNA topics including routing, switching, security, and wireless.

## 📁 Projects

| # | Project | Topics Covered |
|---|---------|---------------|
| 01 | [Extended ACL](./01-extended-acl/) | Traffic filtering, security policies |
| 02 | [Console Configuration](./02-console-configuration/) | CLI setup, device management |
| 03 | [EtherChannel](./03-etherchannel/) | Link aggregation, LACP/PAgP |
| 04 | [HSRP](./04-hsrp/) | Gateway redundancy, failover |
| 05 | [Attack Mitigation](./05-attack-mitigation/) | Network security, threat prevention |
| 06 | [OSPF](./06-ospf/) | Dynamic routing, link-state protocol |
| 07 | [SLAAC & DHCPv6](./07-slaac-dhcpv6/) | IPv6 addressing, stateless config |
| 08 | [WLC](./08-wlc/) | Wireless LAN Controller, Wi-Fi management |
| 09 | [Final Project](./09-final-project/) | Full enterprise network design |

## 🛠️ Tools Used
